# play-store-screenshots

AI-powered Google Play Store screenshot generator for Android apps. A Claude Code skill that scaffolds a Next.js project, designs advertisement-style slides tailored to your brand, and exports PNGs at every Play Store-required resolution.

## What you get

- **Phone screenshots** rendered inside a pre-measured Pixel 8 Pro frame at 1080×1920, 1440×2560, and 1080×2160.
- **7" tablet** and **10" tablet** screenshots with a shared CSS device frame at all Play Store-recommended sizes.
- **Feature Graphic** (1024×500) — the required Play Store listing banner.
- **Multi-language support** — generate screenshots in any number of locales, with RTL-aware layouts for Arabic, Hebrew, Persian, and Urdu.
- **Theme presets** — swap art direction (clean-light, dark-bold, warm-editorial, or anything you define) with one click.
- **Play Store compliance baked in** — every export is flattened to RGB (no alpha channel), size-checked (under 8 MB), and aspect-ratio validated so Play Console accepts the upload on the first try.

## Install

Recommended — add the skill to Claude Code via npx:

```bash
npx skills add hardikamal/play-store-screenshots
```

Install globally across all projects:

```bash
npx skills add hardikamal/play-store-screenshots -g
```

Or clone manually:

```bash
git clone https://github.com/hardikamal/play-store-screenshots.git ~/.claude/skills/play-store-screenshots
```

## Usage

In Claude Code, just describe what you want:

> "Build Play Store screenshots for my habit tracker. 6 slides, clean/minimal style, warm neutrals, premium feel. Include the Feature Graphic."

Claude will ask about your app icon, screenshots, brand colors, feature list, and style direction, then scaffold a Next.js project and generate the slides.

## Export specifications

| Form factor | Recommended size | Aspect ratio |
|-------------|-----------------|--------------|
| Phone | 1080×1920 or 1440×2560 | 9:16 |
| 7" Tablet | 1440×1920 | 3:4 |
| 10" Tablet | 1920×2560 | 3:4 |
| Feature Graphic | 1024×500 (fixed) | ~2:1 |

All screenshots are exported as RGB PNGs (no alpha), under 8 MB, and within Play Store's 16:9–9:16 aspect ratio bounds.

## Tech stack

Scaffolds a project using:
- **Next.js 14+** (App Router)
- **TypeScript** (strict mode)
- **Tailwind CSS**
- **html-to-image** for faithful PNG export
- **React**

The skill itself is a single `SKILL.md` plus a pre-measured `mockup.png` Pixel 8 Pro frame. The frame is generated by `scripts/generate_mockup.py` (included) and verified by pytest.

## Regenerating the mockup

If you want to change the bundled Pixel frame:

```bash
pip install pillow pytest
python scripts/generate_mockup.py
pytest scripts/test_generate_mockup.py -v
```

If you modify `MK_W` or `MK_H`, update the matching constants in `SKILL.md` under "Phone Mockup Component".
